Project Description

Our project aims to show the decision-making process of an AI chess player. We will visualize the evaluation of potential moves by shading the chess board while the AI searches for the move with the best performance value. This will show the user how a chess AI thinks and how the length of evaluation changes as the game progresses.

Project Goals

  • Develop chess rules engine
  • Develop game UI
  • Implement min-max search algorithm
  • Implement alpha beta pruning
  • Implement visualization of decision-making

Decision-making Representation

Potential moves will be displayed and the board positions corresponding to those moves will be shaded based on the performance metric.
